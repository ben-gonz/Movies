# 🎬 Movies: What Makes a Blockbuster?

This repository contains a data analysis project exploring what factors contribute to a successful blockbuster movie.

The project combines and cleans data from multiple sources, including:
- [Rotten Tomatoes](https://www.kaggle.com/datasets/stefanoleone992/rotten-tomatoes-movies-and-critic-reviews-dataset)
- [TMDB (The Movie Database)](https://www.kaggle.com/datasets/asaniczka/tmdb-movies-dataset-2023-930k-movies)
- [The Academy Awards (Oscars)](https://www.kaggle.com/datasets/unanimad/the-oscar-award)

The cleaned and merged dataset is used to power a dashboard that answers the central question: What makes a great blockbuster movie?
